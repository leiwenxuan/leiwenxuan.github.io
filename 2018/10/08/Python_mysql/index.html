一：连接Mysql数据库

1.1连接条件：

  1. 首先有个启动的mysql数据
  2. 有连接数据库的账号和密码
  3. 有权限操作数据库

1.2使用pymysql

import pymysql  #导入模块
conn = pymysql.connect(
    host='127.0.0.1',  #连接地址
    port='3306',  #端口
    user='root',  #mysql用户名
    password='123',  #密码
    database='db1008',  #数据库
    charset='utf8'  #编码方式
)
cursor = conn.cursor()  #获取光标对象，才可以操作数据库
cursor.execute('show tables;')  #执行sql操作
cursor.close()  #关闭光标
conn.close()  #关闭连接

二：使用pymsql增删改查

2.1增删改（用到的操作相似）

import pymysql
conn = pymysql.connect(
    host='127.0.0.1',
    port=3306,
    user='root',
    password='123',
    database='db1008',
    charset='utf8'
)
cursor = conn.cursor()
name = input('input_name:')
pwd = input('inputZ_pwd:')

#增加操作
sql = 'insert into userinfo(username, possword) values(%s, %s);'
cursor.execute(sql,[name, pwd])  #防止SQL注入问题
conn.commit()  #更新数据库 否则添加失败
conn.rollback() 回滚取消操作。不进行添加

#删除操作
sql = 'delete from userinfo where username=%s and possword=%s'
cursor.execute(sql,[name, pwd])  #防止SQL注入问题

#改操作
sql = 'update userinfo set username=%s  where  possword=%s'
cursor.execute(sql,[name, pwd])  #防止SQL注入问题
cursor.close()
conn.close()

2.1.1 获取插入数据的ID(关联操作时会用到)

import pymysql
conn = pymysql.connect(
    host='127.0.0.1',
    port=3306,
    user='root',
    password='123',
    database='db1008',
    charset='utf8'
)
cursor = conn.cursor()
name = input('input_name:')
pwd = input('inputZ_pwd:')

sql = "insert into userinfo values(3,%s,%s);"
cursor.execute(sql, [name, pwd])
conn.commit()

id = cursor.lastrowid
print(id)
cursor.close()
conn.close()

2.2.1 进阶用法 移动光标

# 可以获取指定数量的数据
cursor.fetchmany(3)

cursor.scroll(1, mode="absolute")
# 光标按照相对位置(当前位置)移动1
cursor.scroll(1, mode="relative")
