异常处理

Oct 02, 2018点击

##1：初始异常

###即便是放在try语句中的内容，在遇到报错之后也会中断这段语句的执行

####1.1 类型异常
l = [‘登录’, ‘注册’]
try:
num = int(input(‘input :’)) #输入：非数字报错 invalid literal for int() with base 10: ‘j’
except ValueError:
print(‘异常：num’)

####1.2列表索引异常
try:
num = int(input(‘input:’))
print(l[num -1]) #输入数字大于2 IndexError: list index out of range
except IndexError:
print(‘异常：索引超标!’)

####1.3多分支异常
try:
num = int(input(‘input: ‘))
print(l[num-1])
except IndexError:
print(‘索引越界！’)
except ValueError:
print(‘类型有误！’)

####1.4多异常
try:
num = int(input(‘input:’))
print(l[num -1])
except (ValueError, IndexError):
print(‘报错！’)

####1.5万能异常

try:
    l = []
    # l[3]
    open('aaa')
except Exception:
    print('异常！')

####1.6 as 异常
try:
l[3] #list index out of range
name #name ‘name’ is not defined
except Exception as e:
print(e) #抛出异常的内容

####1.7 多异常加万能异常
try:
num = int(input(‘input :’))
print(l[num -1])
except (ValueError, IndexError):
print(‘异常！’) #持性了这里下面的将不在执行
except Exception as e:
print(e)

##2：异常的其他处理机制

####2.1 else 机制
try:
print(‘My name is Dan’)
except ValueError:
print(‘name is leiwenxuan’)
else:
print(‘出错了’) #报错将不执行, 没触发异常将执行

###2.2finally

try:
    name #异常
    print('hello !')
except NameError:
    print('报错！')
finally:
    print('你好！') #无论异常处不触发都执行
# finally 无论如何都要执行
# 收尾工作，打开了一个文件，占用了一个网络资源，打开了一个和数据库的链接

try:
    f = open('content')
    name #即使报错了，也执行finally
finally:
    f.close()
    print('执行了！')

###2.3 强制异常
raise NameError(‘name’)
raise NameError
